java-对象和类

发表于 2017-10-30 | 分类于 java | | 阅读次数: | 浏览
对象和对象变量

对象和对象变量之间存在着很重要的区别，一个对象变量并没有实际包含一个对象．看下面实例:

Date time = new Date();

其中对象变量time可以引用Date类型的变量，即time只是一个引用而已，new Date()会产生一个对象实例，time存储的是这个对象的引用．任何对象变量的值都是对存储在另外一个地方的一个对象的引用.

构造器

构造器与类同名，构造器在实例化对象时自动执行，不能对一个已经存在的对象调用构造器来达到重新设置实例域的目的．构造器没有返回值．

隐式参数和显式参数

隐式参数时出现在方法名前的类对象，而显式参数是方法中的传递的参数．在每一个方法中，this表示隐式参数．final修饰的实例域，在构建对象时，必须对其实例化．

静态域和静态方法

通过关键字static修饰的变量或者方法被所有的实例化对象所共享，如果将域修饰为static，每个类中只有一个这样的域，而每一个对象对于这样的实例域都有自己的一份拷贝．
静态方法是一种不能向对象实施操作的方法，可以认为静态方法是没有this参数的，它属于类，不属于任何一个对象，例如Math类的方法就属于静态方法，使用这类方法时，如下:

Math.sqrt()

下面的代码演示静态方法:

package com.ctw.javacore;

public class StaticTest {
public static void main(String[] args)
{
    Employer[] staff = new Employer[3];
    staff[0] = new Employer("c", 1001);
    staff[1] = new Employer("t", 1002);
    staff[2] = new Employer("w", 1003);
    for(Employer e:staff)
    {
        e.setid();
        System.out.println("name= "+e.getName()+",id= "+e.getId()+",salary="
                +e.getSal());
    }
    int n = Employer.getNextId();
    System.out.println("Next available id = "+n);
    }
}
class Employer
{
private static int nextId = 1;
private double salary;
private int id;
private String name;
public Employer(String nString , double s)
{
    this.name   = nString;
    this.salary = s;
    id = 0;
}
public String getName()
{
    return name;
}
public double getSal()
{
    return salary;
}
public int getId()
{
    return id;
}
public void setid()
{
    id = nextId;
    nextId++;
}
public static int getNextId()
{
    return nextId;
}
public static void main(String[] args)
    {
    Employer employer = new Employer("Hary", 5000);
    System.out.println(employer.getName()+" "+employer.getSal());
    }
}

从上述代码可以看到，对于静态方法,直接使用类名调用的．
