java.Math类相关方法

发表于 2017-10-08 | 分类于 java | | 阅读次数: | browse
  1. java中对象和对象引用关系,对象和对象引用不是一个概念，对于下面的例子而言
    A a1 = new A();
    它代表A是类，a1是引用，a1不是对象，new A()才是对象，a1引用指向new A()这个对象。在JAVA里，“=”不能被看成是一个赋值语句，它不是在把一个对象赋给另外一个对象，它的执行过程实质上是将右边对象的地址传给了左边的引用，使得左边的引用指向了右边的对象。JAVA表面上看起来没有指针，但它的引用其实质就是一个指针，引用里面存放的并不是对象，而是该对象的地址，使得该引用指向了对象。在JAVA里，“=”语句不应该被翻译成赋值语句，因为它所执行的确实不是一个赋值的过程，而是一个传地址的过程.

  2. 包装类相关概念．包装类的出现是为了然基本类型具备对象的特性，能够调用方法，更加贴合面向对象的思想．
    | 基本数据类型 | 包装类 |
    |——–|——–|
    | byte | Byte |
    | boolean | Boolean |
    | short | Short |
    | char | Char |
    | int | Int |
    | long | Long |
    | float | Float |
    | double | Double |
    因为Byte,Integer,Long对在-128～127之间的数据进行了缓存，如果待转换的数值在该范围内，那么直接进行读取，如果超出范围，那么将会new Integer(),创建对象．而Float和Double,没有该机制，所以每次都是new一个对象．
    在使用equals和＂==＂进行判断时，在-128~127直接的数值，两者结果一样，超过该范围，两者结果不一样，因为”==”比较的是对象的地址，而equals比较的是内容

  3. 常见的Math＆Number方法:

    public class Test {
public static void main(String[] args)
{
Integer x = 5;
x += 5;
System.out.println(x);
System.out.println("90度的正弦值："+Math.sin(Math.PI/2));
System.out.println("0度的余弦值："+Math.cos(0));
System.out.println(Math.ceil(10.5));//向上取整
System.out.println(Math.floor(10.5));//向下取整
System.out.println("π的角度数"+Math.toDegrees(Math.PI));
//Number类方法.
//1.xxxvalue():将Number类对象转换成对应的数据类型的值并返回
Integer number1 = new Integer(7);//装箱,拆箱过程
System.out.println(number1.intValue());
//2.compareTo():将number对象与参数对比,相等返回０，小于参数返回－１，大于参数返回１
System.out.println(number1.compareTo(5));
//3.equals():判断number对象与参数是否相等,相等返回true,不相等返回false.
System.out.println(number1.equals(6));
//4.valueof():返回一个number对象指定的内置数据类型.这里证明number1只是一个对象引用，而不是一个对象.
System.out.println(number1.valueOf(7));
//5.toString():以字符串的形式返回值
System.out.println(number1.toString());
//6.parseInt():将字符串解析为int类型
String a = "-123";
System.out.println(Integer.parseInt(a));
//7.abs():返回参数绝对值
System.out.println(Math.abs(Integer.parseInt(a)));
//8.ceil():对整形变量向左取整，即向上取整
System.out.println(Math.ceil(9.3));
//9.floor():对整形变量向右取整，即向下取整
System.out.println(Math.floor(9.3));
//10.rint():返回与参数最接近的整数,返回值为double
System.out.println(Math.rint(9.3));
//11.round():返回一个最接近的int,long型值
System.out.println(Math.round(9.3));
//12.min():返回两个参数中的最小值
System.out.println(Math.min(9.3,9.2));
//13.max():返回两个参数中的最大值
System.out.println(Math.max(9.3,9.2));
//14.exp():返回自然数底数的e的参数次方
System.out.println(Math.exp(3));
//15.log():返回参数的自然数底数的对数值
System.out.println(Math.log(8));
//16.pow():返回第一个的第二个数的参数次方
System.out.println(Math.pow(2,3));
//17.sqrt():求参数的算术平均根
System.out.println(Math.sqrt(8));
//18.sin():求指定double类型参数的正弦值
System.out.println(Math.sin(Math.PI/4));
//19.cos():求指定double类型参数的余弦值
System.out.println(Math.cos(Math.PI/4));
//20.tan():求指定double类型参数的正切值
System.out.println(Math.tan(Math.PI/4));
//21.toDegrees():将参数转换为角度
System.out.println(Math.toDegrees(Math.PI/3));
//22.toRadians():将角度转换为弧度
System.out.println(Math.toRadians(Math.PI/21));
//23.random():返回一个随机数,下例产生10-13的随机数
System.out.println(Math.random()*3+10);
    }
}
